## 🎓 Quick Tips

### For Beginners:
1. Start with Large Cap
2. Look for 4-5 star ratings
3. Choose low expense ratio
4. Start with min SIP

### For Search:
- Use fund house (HDFC, ICICI)
- Use fund type (Bluechip, Midcap)
- Use partial names

### For Comparison:
- Compare same category
- Check 3Y returns
- Consider expense ratio
- Match your budget

## 🔧 Troubleshooting

### Slow Loading?
- Normal for first load (10-15 sec)
- Cached after that (instant)

### No Real-time Data?
- Check if fetcher file exists
- Platform falls back to static
- All features still work

### Old NAV?
- Click refresh button
- NAV updates daily at 9 PM IST
