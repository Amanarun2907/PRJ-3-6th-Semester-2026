FLOWCHART 1: AGENTIC AI PIPELINE
================================================================

TITLE: Multi-Agent AI Framework for Investment Decision-Making
       (End-to-End Pipeline)

----------------------------------------------------------------
STEP 1: USER INPUT
----------------------------------------------------------------
Box Label: USER QUERY INPUT
Content:
  - User types a natural language investment query
  - Example queries:
      "Which stocks should I buy today?"
      "Is the market bullish or bearish right now?"
      "Give me a complete weekly market intelligence report"
  - Input is a free-text string
  - No structured format required from the user

Arrow: DOWN to Step 2

----------------------------------------------------------------
STEP 2: QUERY ROUTER
----------------------------------------------------------------
Box Label: QUERY ROUTER (FastAPI Backend)
Content:
  - Endpoint: POST /api/agentic/run
  - Receives the user query as input
  - Identifies which agents to activate (default: all 6)
  - Distributes the query to each selected agent
  - Manages the execution pipeline

Arrow: DOWN to Step 3

----------------------------------------------------------------
STEP 3: PARALLEL AGENT EXECUTION
----------------------------------------------------------------
Label: 6 SPECIALIST AGENTS RUN IN PARALLEL

[This step should be shown as 6 boxes side by side or in 2 rows of 3]

--- AGENT 1: STOCK INTELLIGENCE AGENT ---
Box Label: Stock Intelligence Agent
Data Source: Yahoo Finance (fast_info API)
Process:
  - Fetches live price for each of 15 NSE stocks
  - Computes RSI (Relative Strength Index, period=14)
  - Computes MACD (12, 26, 9 parameters)
  - Computes Bollinger Bands (period=20)
  - Compares current price vs 50-day moving average
  - Assigns signal: BUY / HOLD / SELL per stock
Output: List of stocks with live price, RSI, MACD signal, BUY/HOLD/SELL

--- AGENT 2: MARKET ANALYSIS AGENT ---
Box Label: Market Analysis Agent
Data Source: Yahoo Finance (^NSEI, ^BSESN via fast_info)
Process:
  - Fetches live NIFTY 50 price and % change
  - Fetches live SENSEX price and % change
  - Calculates sector-wise % change for 8 sectors
    (Banking, IT, Energy, FMCG, Auto, Pharma, Metals, Telecom)
  - Identifies best-performing and worst-performing sector
Output: Index values, sector heatmap, market mood

--- AGENT 3: SMART MONEY AGENT ---
Box Label: Smart Money Tracker Agent
Data Source: NSE India Official API
  (https://www.nseindia.com/api/fiidiiTradeReact)
Process:
  - Calls NSE official API to get today's FII/DII data
  - Extracts FII net flow (buy - sell) in crores
  - Extracts DII net flow (buy - sell) in crores
  - Generates signal:
      FII > 1000 Cr AND DII > 1000 Cr  -->  STRONG BUY
      FII > 0 AND DII > 0              -->  BUY
      FII < -1000 Cr AND DII < -1000   -->  STRONG SELL
      FII < 0 AND DII < 0              -->  SELL
      Otherwise                        -->  MIXED
Output: FII net, DII net, institutional signal

--- AGENT 4: NEWS & SENTIMENT AGENT ---
Box Label: News and Sentiment Agent
Data Source: Live RSS Feeds
  - Google Finance RSS (Indian market news)
  - Economic Times Markets RSS
Process:
  - Fetches latest 8 headlines from each feed
  - Applies VADER (Valence Aware Dictionary for Sentiment Reasoning)
    to each headline
  - Scores each headline: -1.0 (very negative) to +1.0 (very positive)
  - Classifies: Positive (>0.05), Negative (<-0.05), Neutral (rest)
  - Calculates average sentiment score across all headlines
  - Determines overall market mood
Output: Headlines with sentiment scores, overall mood (Positive/Negative/Neutral)

--- AGENT 5: RISK MANAGEMENT AGENT ---
Box Label: Risk Management Agent
Data Source: Yahoo Finance (1-year historical data)
Process:
  - Downloads 1-year daily closing prices for 10 stocks
  - Computes daily returns for each stock
  - Calculates annualised volatility = std(returns) x sqrt(252) x 100
  - Calculates Value at Risk at 95% confidence:
      VaR(95%) = 5th percentile of daily returns x 100
  - Builds equal-weight portfolio returns
  - Calculates portfolio-level volatility and VaR
  - Computes average pairwise correlation between stocks
Output: Per-stock volatility, VaR, portfolio risk metrics, correlation

--- AGENT 6: ADVANCED ANALYTICS AGENT ---
Box Label: Advanced Analytics Agent
Data Source: Yahoo Finance (10-day and 1-month history)
Process:
  - Fetches 10-day volume data for 15 stocks
  - Computes volume ratio = today's volume / average volume
  - Flags stocks with volume ratio > 1.5x as unusual activity
  - Classifies: Bullish Breakout (high volume + price up)
                Bearish Breakdown (high volume + price down)
  - Computes 1-week and 1-month sector momentum
  - Identifies sectors with accelerating vs decelerating momentum
Output: Volume anomalies, sector rotation signals, momentum scores

Arrow from each agent: DOWN to Step 4

----------------------------------------------------------------
STEP 4: AI ANALYSIS PER AGENT (Groq LLM)
----------------------------------------------------------------
Box Label: GROQ LLAMA 3.3 70B -- AI Analysis Per Agent
Content:
  - Each agent's live data is formatted into a structured prompt
  - Prompt includes: user query + live data + domain context
  - Sent to Groq API (model: llama-3.3-70b-versatile)
  - Temperature: 0.55 (balanced between creativity and accuracy)
  - Max tokens: 280 per agent
  - Returns: 3-4 sentence expert analysis for that domain
  - If API key not available: returns data-only output without AI text

Arrow: DOWN to Step 5

----------------------------------------------------------------
STEP 5: AGENT REPORTS COLLECTION
----------------------------------------------------------------
Box Label: 6 AGENT REPORTS COLLECTED
Content:
  - Each report contains:
      Agent name and domain
      Live data fetched (prices, flows, headlines, etc.)
      AI-generated analysis text
      Status: complete / error
  - All 6 reports stored in memory for synthesis

Arrow: DOWN to Step 6

----------------------------------------------------------------
STEP 6: MASTER REPORT SYNTHESIS
----------------------------------------------------------------
Box Label: MASTER REPORT AGENT -- Groq Llama 3.3 70B
Content:
  - All 6 agent analyses are combined into one master prompt
  - Master prompt instructs the LLM to synthesise findings
  - Temperature: 0.55, Max tokens: 750
  - LLM generates a structured investment report with 6 sections:
      Section 1: Executive Summary (overall market verdict)
      Section 2: Top 3 Investment Opportunities (specific stocks/sectors)
      Section 3: Key Risks to Watch (3 specific risks from data)
      Section 4: Smart Money Signal (FII/DII interpretation)
      Section 5: Recommended Action Plan (3 concrete steps)
      Section 6: Confidence Level (High/Medium/Low with reason)

FLOWCHART 2: SENTIMENT ANALYSIS PIPELINE
================================================================

TITLE: News and Sentiment Analysis Pipeline
       (End-to-End Methodology)

----------------------------------------------------------------
STEP 1: DATA COLLECTION
----------------------------------------------------------------
Box Label: LIVE NEWS COLLECTION
Content:
  - Source 1: Google Finance RSS Feed
      URL: news.google.com/rss/search?q=indian+stock+market+NSE+BSE
      Language: English, Region: India
      Fetches: Latest 8 headlines
  - Source 2: Economic Times Markets RSS Feed
      URL: economictimes.indiatimes.com/markets/rssfeeds/1977021501.cms
      Fetches: Latest 8 headlines
  - Total: Up to 16 headlines per run
  - Data extracted per article:
      Title (headline text)
      Summary (brief description)
      Source name
      Published timestamp
      Article URL

Arrow: DOWN to Step 2

----------------------------------------------------------------
STEP 2: TEXT PRE-PROCESSING
----------------------------------------------------------------
Box Label: TEXT PRE-PROCESSING
Content:
  - Input: Raw headline text
  - Operations performed:
      Remove special characters and HTML tags
      Normalize whitespace
      Convert to lowercase for analysis
      Extract key financial terms
  - No stopword removal (VADER handles this internally)
  - Each headline treated as an independent document

Arrow: DOWN to Step 3

----------------------------------------------------------------
STEP 3: DUAL SENTIMENT SCORING
----------------------------------------------------------------
Box Label: DUAL SENTIMENT ENGINE
Content:
  Two tools applied to each headline:

  TOOL A: VADER (Valence Aware Dictionary and sEntiment Reasoner)
    - Specifically designed for short social media and news text
    - Uses a lexicon of 7,500+ words with sentiment scores
    - Handles punctuation, capitalization, and intensifiers
    - Returns compound score: -1.0 (most negative) to +1.0 (most positive)
    - Also returns: positive, negative, neutral proportions
    - Example:
        "RBI cuts interest rates, market rallies"
        --> compound score: +0.72 (Positive)
        "FIIs sell Rs 5000 Cr, market crashes"
        --> compound score: -0.68 (Negative)

  TOOL B: TextBlob
    - Linguistic analysis using pattern-based approach
    - Returns polarity: -1.0 to +1.0
    - Returns subjectivity: 0.0 (objective) to 1.0 (subjective)
    - Useful for detecting opinion vs factual statements

  Final score = VADER compound score (primary)
  TextBlob used as secondary validation

Arrow: DOWN to Step 4

----------------------------------------------------------------
STEP 4: SENTIMENT CLASSIFICATION
----------------------------------------------------------------
Box Label: SENTIMENT CLASSIFICATION
Content:
  Based on VADER compound score:
    score > +0.05   -->  POSITIVE  (bullish signal)
    score < -0.05   -->  NEGATIVE  (bearish signal)
    -0.05 to +0.05  -->  NEUTRAL   (no clear signal)

  Each article gets:
    - Sentiment label (Positive / Negative / Neutral)
    - Numeric score (-1.0 to +1.0)
    - Confidence of classification

Arrow: DOWN to Step 5

----------------------------------------------------------------
STEP 5: SECTOR MAPPING
----------------------------------------------------------------
Box Label: SECTOR-WISE SENTIMENT MAPPING
Content:
  Each headline is mapped to a market sector using keyword matching:
    Banking:  keywords = [bank, rbi, npa, credit, loan, hdfc, icici, sbi]
    IT:       keywords = [tcs, infosys, wipro, it sector, software, tech]
    Pharma:   keywords = [pharma, drug, fda, medicine, healthcare]
    Auto:     keywords = [auto, car, ev, electric vehicle, maruti]
    Energy:   keywords = [oil, gas, reliance, ongc, energy, power]
    FMCG:     keywords = [fmcg, consumer, hul, itc, nestle]
    Metals:   keywords = [steel, metal, tata steel, hindalco, copper]
    IPO:      keywords = [ipo, listing, grey market, gmp, subscription]
    General:  (default if no sector keyword matched)

  Output: Each article tagged with sector label

Arrow: DOWN to Step 6

----------------------------------------------------------------
STEP 6: AGGREGATION
----------------------------------------------------------------
Box Label: SENTIMENT AGGREGATION
Content:
  Article-level --> Sector-level:
    - Group all articles by sector
    - Calculate average sentiment score per sector
    - Count Positive, Negative, Neutral articles per sector
    - Determine sector sentiment label:
        avg > +0.05  -->  Positive (bullish sector news)
        avg < -0.05  -->  Negative (bearish sector news)
        otherwise    -->  Neutral

  Article-level --> Market-level:
    - Calculate overall average sentiment across all articles
    - Determine overall market mood
    - Count total Positive, Negative, Neutral articles

Arrow: DOWN to Step 7

----------------------------------------------------------------
STEP 7: AI INTERPRETATION (Groq LLM)
----------------------------------------------------------------
Box Label: AI INTERPRETATION -- Groq Llama 3.3 70B
Content:
  - All headlines with their sentiment scores are passed to Groq LLM
  - Prompt includes:
      All headlines with [Positive/Negative/Neutral] labels
      Overall sentiment score
      User's original query
  - LLM generates:
      Key themes in today's news
      How sentiment affects investment decisions
      Sectors to watch based on news
      Short-term market outlook from news perspective
  - Output: 3-4 sentence expert news analysis

Arrow: DOWN to Step 8

----------------------------------------------------------------
STEP 8: VISUALIZATION OUTPUT
----------------------------------------------------------------
Box Label: SENTIMENT VISUALIZATION
Content:
  Four visual outputs generated:

  Chart 1: Sentiment Distribution Pie Chart
    - Shows count of Positive, Negative, Neutral articles
    - Colors: Green (Positive), Red (Negative), Orange (Neutral)

  Chart 2: Sector Sentiment Bar Chart
    - X-axis: Sectors (Banking, IT, Pharma, etc.)
    - Y-axis: Average sentiment score
    - Green bars = positive sector news
    - Red bars = negative sector news

  Chart 3: Sentiment Timeline Chart
    - X-axis: Article index (latest to oldest)
    - Y-axis: Sentiment score
    - Shows trend of market mood over recent articles

  Chart 4: News Feed with Sentiment Labels
    - Each headline displayed with color-coded sentiment badge
    - Clickable links to original articles
    - Source and timestamp shown

Arrow: DOWN to Step 9

----------------------------------------------------------------
STEP 9: INTEGRATION WITH AGENTIC AI
----------------------------------------------------------------
Box Label: INTEGRATION WITH AGENTIC AI PIPELINE
Content:
  - Sentiment output feeds into the News & Sentiment Agent
  - Agent passes sentiment data to Master Report Agent
  - Master Report uses sentiment to:
      Validate or contradict technical signals
      Adjust confidence level of recommendations
      Identify news-driven vs fundamentals-driven moves
  - Example:
      If RSI says BUY but sentiment is strongly Negative
      --> Master Report flags: "Technical BUY but news headwinds"
      --> Confidence level reduced to Medium

Arrow: DOWN to Step 10

----------------------------------------------------------------
STEP 10: FINAL SENTIMENT OUTPUT TO USER
----------------------------------------------------------------
Box Label: FINAL OUTPUT TO USER
Content:
  - Overall market mood: Positive / Negative / Neutral
  - Sector-wise sentiment heatmap
  - Top 3 positive news themes
  - Top 3 negative news themes
  - AI-generated news analysis paragraph
  - Sentiment score: -1.0 to +1.0 (displayed as gauge)
  - Recommendation adjustment based on sentiment
